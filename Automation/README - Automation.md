# **Selenium Automation Testing Project**

## **Project Overview**

This project demonstrates automated UI testing using Selenium WebDriver, Java, and TestNG.

The framework automates core user workflows and validates functionality through automated assertions.

## **Technologies Used**

* Java  
* Selenium WebDriver  
* TestNG  
* Maven  
* EdgeDriver

## **Automated Test Scenarios**

* User login  
* Invalid login validation  
* Add to cart  
* Checkout flow  
* Form validation

## **Framework Features**

* TestNG test execution  
* Assertions and validations  
* Reusable page objects  
* Organized test structure  
* Automated browser interaction

## **Included Artifacts**

* Automation framework source code  
* Test plan  
* Execution screenshots  
* Test execution reports

## **How to Run Tests**

1. Clone repository  
2. Install dependencies using Maven  
3. Run testng.xml
