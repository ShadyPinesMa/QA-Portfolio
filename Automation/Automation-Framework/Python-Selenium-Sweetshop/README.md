# Sweetshop Python Selenium Test

Automated UI test for https://sweetshop.vivrichards.co.uk/ written in Python using Selenium Webdriver and pytest.

This test is a port of an existing test from my Java/TestNG Selenium framework and is meant to show language transfer from Java to Python

## Tech Stack

- Python 3
- Selenium WebDriver
- pytest

## How to Run

1. Install dependencies: 'pip install selenium pytest'
2. Run test: 'pytest test_sweetshop.py'
