# **Manual Testing Project**

## **Project Overview**

This project demonstrates manual testing practices performed on the [bookshop.org](http://bookshop.org) web application using structured test planning, test case design, and defect reporting.

## **Scope**

The testing focused on:

* Login functionality  
* Search functionality  
* Form validation

## **Testing Types Performed**

* Functional Testing  
* Smoke Testing  
* Positive Testing  
* Negative Testing  
* Exploratory Testing

## **Tools Used**

* Google Sheets  
* Google Docs  
* Jira concepts for defect tracking

## **Included Artifacts**

* Test Plan  
* Manual Test Cases  
* Bug Reports  
* Screenshots

## **Key Skills Demonstrated**

* Test case design
